# Reload on Breakpoints
A simple script that reloads when your Site is passing a defined breakpoint.

## Installation
Install via NPM:

```bash
npm install reload-on-breakpoint

```

## Usage

#### javascript

```javascript

const bg = new ReloadOnBreakpoint([768, 992, 1200, ...]);

bg.mount();

```
